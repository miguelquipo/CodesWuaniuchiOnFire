Las funciones moveBackward()y turnRight() son dos funciones que controlan el movimiento de un carrito o robot utilizando dos motores, "MOTOR 1" y "MOTOR 2".

La función moveBackward() esta función se encuentra diseñada para hacer que un carrito se mueva hacia atrás. Al igual que las funciones anteriores, esta función controla dos motores, "MOTOR 1" y "MOTOR 2", para lograr el movimiento hacia atrás.

La función turnRight() esta función se encuentra diseñada para hacer que un carrito gire hacia la derecha y avance al mismo tiempo. Ademas, esta función controla dos motores, "MOTOR 1" y "MOTOR 2", y también llama a la función moveForward()para lograr el movimiento adelante hacia mientras gira a la derecha. 

    void moveBackward() {        //HACIA ATRAS
        // MOTOR 1 
        digitalWrite(ENA, 150);
        digitalWrite(LM1, HIGH);
        digitalWrite(LM2, LOW);
        // MOTOR 2
        digitalWrite(ENB, 150);
        digitalWrite(RM3, HIGH);
        digitalWrite(RM4, LOW);    
    }  
    void turnRight() {     //GIRA A LA DERECHA Y AVANZA
        // MOTOR 1
        analogWrite (ENA, 150);
        digitalWrite(LM1, HIGH);
        digitalWrite(LM2, LOW);
        // MOTOR 2
        analogWrite (ENB, 150);
        digitalWrite(RM3, LOW);
        digitalWrite(RM4, HIGH);
        moveForward();
    }

La función turnLeft() esta función se encuentra diseñada para hacer que el carrito gire hacia la izquierda y avance al mismo tiempo. Esta función controla dos motores, "MOTOR 1" y "MOTOR 2", y también llama a la función moveForward()para lograr el movimiento adelante hacia mientras gira a la izquierda. 

    void turnLeft() {           //GIRA A LA IZQUIERDA Y AVANZA
